[编辑] Ubuntu 意味着什么?

Ubuntu 是一个非洲词汇，它的意思是“人性对待他人”或“群在故我在”或者是“博爱”。Ubuntu 发行版将 Ubuntu 的精神带到软件世界中。

Ubuntu基于十分稳定的发行版Debian，如果要追求稳定，可以尝试Debian Stable或者Debian Testing。
