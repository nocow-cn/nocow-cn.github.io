[编辑] 系统

Powertop

  • Powertop是Intel专门为Intel cpu推出的一款在Linux下运行的节电程序。可以找出常常把cpu从闲置中唤醒的程序，然后修复或者关闭那些进程，达到省电的目的。PowerTOP 要求 Linux 内核开启 tickless idle 功能，并具备 2.6.21 及以上版本，所以只能在ubuntu 7.10下使用。。
