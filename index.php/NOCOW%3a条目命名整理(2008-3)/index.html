NOCOW:条目命名整理(2008-3)

来自NOCOW
跳转到： 导航, 搜索

2008年3月22日开始对条目的命名进行了一些整理。

USACO题解已经全部放在USACO/题目名，翻译在Translate:USACO/题目名，代码在Translate:USACO/题目名/语言（首字母大写）。

对于此命名规则的方便的条目查找方法见NOCOW:MediaWiki程序修改#搜索框修改。

为了打开Code:名字空间，以前以Code:开头的条目已经全部移动到新名字空间并把原来的重定向页面删除。为此可能会有一些链接失效，请大家谅解。我已经用程序自动更新了USACO题解中的链接，如果发现有什么问题希望大家能帮助修改一下。

近期可能还会有有其他的整理。包括可能会删除一些陈旧的链接，启用更多的名字空间等等。

#!/usr/bin/expect -f
 
set cookiefile [if {$argc == 0} {expr {{cookie}}} {lindex $argv 0}]
 
log_user 0
 
spawn wget -O- --load-cookies=$cookiefile --save-cookies=$cookiefile --keep-session-cookies "http://www.nocow.cn/index.php?title=-&action=edit"
 
expect -indices -re {<input type='hidden' value="([0-9a-f]*)\\" name="wpEditToken" />}
set res $expect_out(1,string)
expect eof
 
if {$argc < 2} {send_user WPEDITTOKEN=$res\n } else {eval exec env WPEDITTOKEN=$res COOKIEFILE=$cookiefile [lrange $argv 1 end]}

[编辑] edit.sh

#!/usr/bin/expect -f
 
log_user 0
set pagename [string map {" " + + "%2B"} [lindex $argv 0]]
set timeout 10000
 
spawn wget -O- --load-cookies=cookie --save-cookies=cookie --keep-session-cookies "http://www.nocow.cn/index.php?title=$pagename&action=edit"
 
set timeout 10000
match_max 100000
 
expect {<form id="editform"}
 
expect -indices -re {<input type='hidden' value="(.*)" name="wpStarttime" />}
set wpStarttime $expect_out(1,string)
expect -indices -re {<input type='hidden' value="(.*)" name="wpEdittime" />}
set wpEdittime $expect_out(1,string)
 
expect {
-indices -re {<textarea tabindex='1' accesskey="," name="wpTextbox1".*>(.*)</textarea>} {
set originaldata $expect_out(1,string)
 
expect -indices -re {<input type='hidden' value="(.*)\\" name="wpEditToken" />}
set wpEditToken $expect_out(1,string)
expect -indices -re {<input name="wpAutoSummary" type="hidden" value="(.*)" />}
set wpAutoSummary $expect_out(1,string)
expect eof
 
#edit
spawn echo [string map {&lt; < &gt; > &quot; \" &amp; %26 & %26 + %2B} $originaldata]
match_max 100000
set submitdata {}
 
expect -re {[[][[]Code:} {
set submitdata $submitdata$expect_out(buffer)
expect -indices -re {([^_ ]*)[_ ]([^|#]*[|#])}
set submitdata ${submitdata}USACO/[string tolower $expect_out(1,string)]/[string totitle $expect_out(2,string)]
exp_continue
}
 
set submitdata $submitdata$expect_out(buffer)
 
send_user ready
 
exec wget --load-cookies=cookie --save-cookies=cookie --keep-session-cookies "http://www.nocow.cn/index.php?title=$pagename&action=submit" "--post-data=wpSection=&wpStarttime=$wpStarttime&wpEdittime=$wpEdittime&wpScrolltop=&wpTextbox1=$submitdata&wpSummary=robot(edit-naming)beta&wpSave=保存此页&wpEditToken=$wpEditToken%5C&wpAutoSummary=$wpAutoSummary"
 
}
-re {<textarea tabindex='1' accesskey="," name="wpTextbox1".*Code:USACO/} {expect eof;send_user unchanged;exit;}
eof {send_user eof;exit;}
}

本代码专门用来更新USACO的链接，如果需要用于其他目的，可以编辑一下代码中#edit以下的部分。
