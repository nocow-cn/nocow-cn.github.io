Fractions to Decimals

分数化小数

译 by tim green

[编辑] 描述

写一个程序，输入一个形如N/D的分数(N是分子，D是分母)，输出它的小数形式。 如果小数有循环节的话，把循环节放在一对圆括号中。

例如, 1/3 =0.33333333　写成0.(3) 41/333 = 0.123123123...　写成0.(123) 用xxx.0 成表示整数 典型的转化例子： 

1/3 = 0.(3)
22/5 = 4.4
1/7 = 0.(142857)
2/2 = 1.0
3/8 = 0.375
45/56 = 0.803(571428)

PROGRAM NAME: fracdec

INPUT FORMAT

单独的一行包括被空格分开的　N和D，　1 <= N,D <= 100000。

[编辑] SAMPLE INPUT

(file fracdec.in)

45 56

OUTPUT FORMAT

小数的表示方法上面说的很明白了，如果输出的长度超过76个字符，每行输出76个。

[编辑] SAMPLE OUTPUT

(file fracdec.out)

0.803(571428)
