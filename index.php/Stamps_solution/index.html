USACO/stamps

来自"NOCOW"

< USACO(重定向自Stamps solution)
跳转到: 导航, 搜索

[编辑] 分析

动态规划的题目，用F[i]表示得到i面值所需要的最少邮票个数，Value[j](j=1..Stamps)表示每个邮票的面值，可得以下转移方程： 

F[i]=Min ( F[i-Value[j]] + 1 ) (i-Value[j]>=0 j=1..Stamps)
初始状态：F[0]=0;F[i]=INFINITE;

如果计算中间某个F[i]大于了最大可以使用的邮票数量，则退出循环，输出i-1即为解

小优化： 初始化时先计算出最大面值的邮票，令其面值为MaxValue，然后在初始化F时可令： 

F[k*MaxValue]=k;

这表示要得到k*MaxValue面值的邮票最少需要k张邮票，这是显然的。于是在计算时我们可以免去(1/MaxValue)*100%的运算量。
