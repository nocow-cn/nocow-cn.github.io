[编辑] 1521. War games 2

Time Limit: 1.0 second

Memory Limit: 16 MB

[编辑] Background

这是一款战争游戏，和苏联有关。细节略~♪

[编辑] Problem

按照计划,这款游戏分成n个部分和 N 战士, 从1到n标号,被排成一圈。每个阶段,一个士兵离开圈去clean the WC,另外人仍然排着队。有些阶段, 出局的那个士兵是上一轮出局士兵之后的第k个.第K个士兵第一轮出圈。

Mr. Rascal知道这些士兵的能力并且对他们能够自己排定顺序不抱有任何希望。 "这些笨蛋甚至不能粉刷草地（？）", - 他对此嗤之以鼻并向sergeant Filcher寻求帮助。

[编辑] Input Format

只有一行包括N (1 ≤ N ≤ 100000) 和 K (1 ≤ K ≤ N).

[编辑] Output Format

你应该在士兵出圈的时候输出他们的编号. 每个数都用空格隔开.

[编辑] Sample Input 

5 3

[编辑] Sample Output 

3 1 5 2 4
