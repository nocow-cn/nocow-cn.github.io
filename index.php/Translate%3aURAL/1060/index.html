[编辑] discription

翻币游戏是在一个4*4的矩形区域的16个格子里放置着双面的硬币，硬币一面黑，一面白，硬币放置时，不是黑面朝上就是白面朝上。每一个步可以翻动3至5个币，从而改变硬币朝上的颜色，使原来黑面朝上变为白面朝上，或者原来白面朝上变为黑面朝上。翻币过程遵循以下规则：

（1）可以选择16个中的任一个来翻

（2）翻动选中的那个币，以及与它上下左右相邻的硬币（假如有的话）

例子： 

bwbw
wwww
bbwb
bwwb

"b"代表黑面朝上，"w"代表白面朝上，假如选择翻动第3行第一个（图中标示的那个） ，那么（翻动后）就会变成为： 

bwbw
bwww
wwwb
wwwb

目标是使所有的硬币黑面朝上或白面朝上。 求出所需的最小步数。

[编辑] Input

输入为4行字符，每行4个字符（ "w" 或"b"），表示游戏区域的（初始）状态。

[编辑] Output

输出只含一个整数 ----从给定的状态到目标状态所需的最小步数。假如初始状态就是目标状态，输出0；假如不可能达到目标状态，输出"Impossible" （不含引号）。

[编辑] Sample Input 

bwbw
wwww
bbwb
bwwb

[编辑] Sample Output

Impossible
