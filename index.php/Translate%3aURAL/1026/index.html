[编辑] Background

在五角大楼的数据库中包含着一个绝密信息。我们不知道那是什么——你知道，那是绝密的，但是我们知道信息的表现方法。它非常的简单。我们不知道为什么，所有的资料被编为了从1到5000的自然数。数据库的大小相当大——它可能容纳了100000个像那样的数字。数据库能够很快的处理每一个问题。最通常的问题是：“排序后第i个值是哪一个？”i是一个范围在1到N之间的自然数。 （按值在从最小的到最大中的顺序）。

[编辑] Problem

你的程序扮演数据库控制器的角色。在其它的命令中，它处理问题应该像那控制器一样快。


[编辑] Input

输入分为的两个部分。首先，是一个数据库，然后是问题的序列。数据库的格式是十分简单的：在第一行有一个数字N(1<=N<=100000)，在以下的N行中是处于任意顺序的每行一个的数据库中的数字。问题的序列很简单：在第一行是问题的总数K（1 <= K <= 100）。然后在以下的K行中，每一行是一个问题。问题“排序后第i个是哪一个？”被编为数字i。数据库和问题的内容的有三个符号"#"分割。

[编辑] Output

输出的内容应该由K行组成。在每一行中应该有一个对于相应问题的答案。

[编辑] Sample Input 

5
7
121
123
7
121
###
4
3
3
2
5

7
