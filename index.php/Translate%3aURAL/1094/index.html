E-screen（电子屏）

[编辑] Discription

一个超级市场里安装了一个新的单用户电子屏，它可以实现信息的快速输入和修改。所有的信息都由一个操作员输入。操作员每按下一个键，它对应的字符就会出现在屏幕的当前光标所在位置上（这个位置上原有的字符被删除），并且光标向右移动一位。

键盘上包含了字母a-z，A-Z，数字0-9，标点符号(:;-!?.,)和空格，还有两个键是表示光标向左和向右移动一格（光标移动时不会删除任何字符）。屏幕的宽度是80字符。当光标到达屏幕的最左或最右端时会自动地跳到左边第一格的位置。

这个新的电子屏在销售商的操作下工作得非常好，但当销售商离开以后就没有人能够适当地操纵这台机器了。而且这个电子屏是被安装在一个操作员看不见的地方。你的任务是设计一个程序效法这个电子屏，使操作员能够看到他/她的操作结果。

[编辑] Input Format

输入：只有一行，即操作员按键的序列。其中符号'>'表示光标向右移动一位，符号'<'表示光标向左移动一位。输入不超过10000个字符。

[编辑] Output Format

输出：当所有按键序列都输入完毕后，电子屏上显示的内容。 假定一开始时电子屏显示80个空格，并且光标放置在左边第一位。

[编辑] Sample Input 

>><<<Look for clothes at the <<<<<<<<<<<<<<<second flo
or. <<<<<<<Fresh pizza and <<<<<<<<<<<<<<<<hamburger a
t a shop right to <<<<<<<<<<<<<the entrance. Call <<<<
<<<<<< 123<-456<-8790 <<<<<<<<<<<<<<<<to order <<<<<<<
<<<<<<<<<<computers< and office<<<<<<< chairs.

[编辑] Sample Output 

Look for second hamburger at computer and chairs.790
