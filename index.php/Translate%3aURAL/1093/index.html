Darts 飞镖

时限：3s

你可能知道一个叫飞镖的游戏。游戏的目标是击中一个叫做镖靶的圆形目标。为了那样你要用一个飞镖投掷镖靶。当然，你需要一些时间学习怎样投掷。因此你的任务是编写一个程序加快学习过程，该程序用于确定一个飞镖能否击中镖靶。 

   我们作出以下假想：

   1）一个镖靶是平面上的一个圆形。它又下列数据定义：圆心（Cx,Cy,Cz）；半径R；一个

与该镖靶所在平面垂直的向量的坐标（Nx,Ny,Nz）。 

   2）一个飞镖是一个按照下列方程移动的质点：

   Mx = Sx + Vx * t

   My = Sy + Vy * t

   Mz = Sz + Vz * t - (g/2)*t2

   这里Sx，Sy和Sz是飞镖的初始位置的坐标，Vx，Vy和Vz是飞镖出速度的分速度，Mx, My和Mz是飞镖在时间t所在的坐标，g是重力加速度，取10m/s2。

   3）可以假设飞镖以非零的水平速度投掷（即Vx或Vy是非零数）。

   4）可以假定没有摩擦力。

   5）击镖耙时飞镖要击中它边界内的区域。

　

输入

输入包括13个数。它们是（根据上述的符号）：

Cx Cy Cz Nx Ny Nz R Sx Sy Sz Vx Vy Vz

输出

如果击中，输出一个词“HIT”（没有双引号），否则输出“MISSED”。

样例输入

47 0 -72 1 0 1 4.25 0 0 0 10 0 10 样例输出

HIT
