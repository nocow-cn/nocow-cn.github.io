[编辑] 1636. Penalty Time 罚时 

Time Limit: 1.0 second
Memory Limit: 64 MB

zzz队又一次输了。他们是第一个做完所有题目的，但是qxx的总时更少。
.....
.....
其实就是要你减掉zzz的罚时，看zzz队的总时会不会比qxx的少，zzz队会不会赢。

[编辑] Input 

第一行 t1,t2(分别是qxx和zzz的总时)
第二行 zzz十道题的罚时次数【每次罚20分钟】

[编辑] Output 

如果qxx还是赢了，输出“No chance.”
否则输出“Dirty debug :(”

[编辑] Samples 

Input1:
290 420
0 0 0 2 1 0 2 0 1 0
Output1:
No chance.
Input2:
300 719
0 0 0 0 0 0 21 0 0 0
Output2:
Dirty debug :(
