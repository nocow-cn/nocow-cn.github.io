1603.“博学” 时间限制：1s 内存限制：64MB

题目背景：

Petr喜爱做横纵连词游戏等谜题。一次，他在报纸中找到了一个新的谜题，名为“博学”：现有一个由英文字母填满的4×4的方格。你需要在这个方格里连出尽可能多的在给定字典中出现的单词；找单词的时候，单词从一个字母开始，之后你可以向上、下、左、右了四个方向连词，连词的过程中你可以拐多次的弯，但是不可以去此次连词中已走过的位置(即不能重复取同一个位置的字母)。 Petr的朋友Vasya告诉他在这样白痴的游戏里花时间是不值得的。Vasya说写一个程序来连词会简单的多。这句话冒犯了Petr，Petr生气地说：“如果你有那么聪明，你就自己写一个程序好了。我会用我喜欢的连词方式和你比赛。” 为了帮助Vasya摆脱这个尴尬的处境，写这个程序的任务就交给聪明的你了。

输入格式

前四行为这个4×4方格，每一个小方格中包含一个小写的英文字母。接下来一行一个数n（n<=100）表示给定字典中包含的单词总数。接下来n行每行为一个长度为1-16的单词，单词仅由小写字母构成。

输出格式

对于字典中的每一个单词，如果能在方格中连出它，输出"YES"，否则输出"NO"，具体格式请参照样例

样例输入

abra adac babr arca 3 abracadabra ababaab ababaaba

样例输出

abracadabra: YES ababaab: YES ababaaba: NO
