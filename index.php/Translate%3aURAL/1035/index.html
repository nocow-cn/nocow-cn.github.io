[编辑] discription

考古学家找到一块刺绣，这块刺绣由一些线组成。规定如下：

那块布有正方形的小格子。

每条缝线覆盖格子的对角线。缝线可以在布的任意一边，但是每一条缝线只在布的一边（线只可以在格子的顶点开始、结束和通过）。

每一条缝线可以在一边的格子的对角线上。

每条线由一些缝线交替安排在布的两边。（一条线的两条连续的缝线在布的两边和在顶点上连接）

一针只能在格子的顶点上通过。

这是一个由6条缝线组成的样例。格子的大小是4*5。布的正面画着上面的形状。在正面的缝线用实线，背面的用虚线。下面的图跟上面一样，正面用虚线，背面用实线。这可以看出缝线在格子的对角线。这块布不会小于4条线。

考古学家想知道这个样例最小由几条线组成。你的程序需要计算出样例所需的最小线数。 　

[编辑] Input Format

第一行是整数 N 和 M ，这表示格子的大小是 N * M （1<= N,M <=200）。之后的是2*N 行，分别表示布的正反两面。“.”表示空格子，测试样例如上。

[编辑] Output Format

输出包含一个整数：最少需要的线的数目。

[编辑] Sample Input 

4 5
.....
.\...
..\..
.....
.....
....\
.\X..
.....

[编辑] Sample Output 

4
