[编辑] 公司历史

TopCoder是在线编程赛事、技能评估以及竞争性软件开发这三个领域的行业先锋。它于2001年成立，旨在世界范围内发掘杰出的计算机软件工程师。客观评价各个软件开发人员的技能是一件很困难的事，于是进行程序锦标赛的想法应运而生。TopCoder提供了一种的在线发布软件需求，接收并测试软件代码的方式，已经获得专利。这种方式已经迅速成为举办程序设计竞赛，收集技术数据的默认平台。TopCoder公司将这个平台进行扩展，举办了更多优秀的比赛并因此开发了很多商业级别的组件和应用软件。

[编辑] 算法竞赛

TopCoder平台（比赛竞技场）是一大特色，它能展示并评定参赛者。他们需要在规定的时间内解决指定的挑战性算法问题，不准作弊。他们可以使用多种编程语言，包括Java、C++和 Visual Basic。参赛者在相同的时间内, 解决相同的问题, 机会是均等的。所提交的代码会由测试数据集进行测试，并迅速客观的给出测试结果。 TopCoder平台包括两个关键部分：客户和服务器。 TopCoder Arena实际上是一个Java程序，全球各地的用户都可轻松下载。服务器则是Linux和Solaris服务器, Apache Web Server 技术, BEA’s Weblogic Application Server 技术以及Informix数据库的共同组合。

[编辑] 社区

TopCoder成员有来自各个领域的软件开发专家，包括工业和教育界的开发者, 你可以免费成为TopCoder会员。以下是一些关键的人员统计信息:

  • 会员总数超过100,000人
  • 会员51％为学生，49％为专业人士
  • 会员来自世界各地190多个国家和地区

[编辑] 商业模式

TopCoder在不同的模式下盈利，收入主要来自赞助TopCoder赛事和软件开发两方面。

[编辑] 赞助TopCoder赛事

TopCoder 每年主办两次国际性的编程比赛：TopCoder大学生挑战赛(TopCoder Collegiate Programming Contest, TCCC)和TopCoder公开赛(TopCoder Open,TCO)。TopCoder大学生挑战赛每年春季在网上进行，决赛现场在世界一流的高等学府现场进行。TopCoder公开赛则是每年秋季在风景名胜地的会议中心举行。为了鼓励TopCoder的优秀成员参与，很多商家为了提升知名度和促进赛事的给予而慷慨赞助。而事实上，很多著名技术公司发现赞助TopCoder竞赛这一独特方式，是抓住国际一流开发者注意力的绝佳途径。

[编辑] 客户

Google,Yahoo!, Sun, Intel, Motorola, Microsoft, Intel Corporation, AMD.

[编辑] 软件

TopCoder软件通过公司顾客和个人开发者获得收入。TopCoder软件组件目录(Software Component Catalogue)是一个底层组件库(用来构造更大系统的通用软件模块)。TopCoder通过与我们赛事中的佼佼者合作来构造我们自己的商业软件，任何开发商都可以购买我们的组件来提高他们员工的开发效率并降低开发成本。此外，我们利用该目录直接为客户构造更大更复杂的系统。

[编辑] 客户

Praxair, ING, ABB, Philip Morris (Altria Group Inc.) Wyeth, ESPN, VeriSign.
