[编辑] 我们是谁

美国计算机奥林匹克竞赛通过培训、比赛和国际竞赛培养大学生计算机预科。有一些教练来组织这些活动，同时也陪伴在国际场合中代表美利坚的学生。详情访问USACO官网。

[编辑] USACO具体目标

USACO的具体目标包括：

  • 训练美国队在下一届IOI中获得更多金牌。
  • 促进美国学生的学习和成长以达成第一个目标。
  • 组织一个由15名学生参赛的八日美国奥赛，从中选出四个来参加国际比赛。

[编辑] 关于高级训练

网站包括一些训练，提供给有一到两年编程经验，且对「算法编程」有一定兴趣的学生。在这里钻研并传授技术确实不易。如果你是一名优秀的程序员，可能每道题目会花费几分钟到若干小时不等的时间。初学者会不太喜欢这些题目，因为他们需要广泛的实践和经验。或许有一天我们会为初学者搭建一个低层次的训练网站。

[编辑] 关于此网站

网站为上面所述之目标而设计制作。

  • 密码受到保护。
  • 所有功能全天24小时可用，所以任何人都可把无压力的空闲时间用于训练。
  • 教学文本、题目和分析尽量简洁，减少阅读时间，提高编程效率。
  • 在接近两打的教学文本针对每个题型中补充了100多道比赛模式的题目。
  • 各个阶段的题目排列取决于难度而非「题型」。
  • 基于网页的排序系统引导学生完成更多题目和文本。
  • 算法解释详细题目IOI化使竞赛者熟悉国际竞赛环境。
  • 任务要求提交程序到自动评测系统，用多组测试数据进行评判。
  • 网站对题目评测提供快速的反馈机制（通常只需几秒）
  • 分析页面阐述题目的解决方法和详细信息，代码优质且附有注释。

[编辑] 获得奖励和特权

USACO将会在你完成所有题目和文本时寄送进步证书。此外，杰出的表现将带给你去六月USACO训练营的机会。

季末小奖励将颁给那些对于复杂问题有着最快解决方案的人，参看网站以查询。

不要被开始的题目迷惑。后面的题目更精彩。

[编辑] 要用多长时间？

这个就因人而异了，尤其是100+编程挑战。一些学生已经有多年经验，可能已经达到国际水平。一些只是刚刚开始参与。两周一道题目是最低限度了。优秀的学生应当在前两节以每天至少一道题目的速度来完成（往后就慢了）。

要在本网站获得更多，你应当按顺序依次解决每个问题，理解教学文本中每一段话，并将其消化吸收用于解题。读完分析后自行改进你的解答。更甚者，还可以自行寄给我们更好地分析和解答。

[编辑] 文案

一些小技巧会使使用变得轻松：

  • 用「返回USACO门口」"Back to USACO Gateway" 链接来查看即时更新
  • 用「提交评论或问题」"Submit a Comment or Question"链接与教练取得联系
  • 查看左栏的「新闻」"NEWS"获取最新消息
