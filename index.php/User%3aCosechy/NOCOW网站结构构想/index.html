[编辑] 主名字空间

  • 题库名
  • 题库名/题目提交id
  • 经典题目名
  • 经典问题类型名
  • 数据结构名
  • 算法名
  • 算法科学分支名
  • 算法及理论分析科学分支名
  • 问题(算法)名
  • 理论分析用数学知识
  • 理论分析及解决问题用数学知识
  • 算法类型
  • 系列竞赛(活动名)
  • 活动名
  • 单一竞赛名称（包括年号）
    • 之前的方案：系列竞赛名(题库名)/(主观的)竞赛名(年号)
  • 书名
  • OI相关组织名
  • 学校名（orz，这个没什么关系，可以不写）
  • OI相关著名人物名
  • 数学或其他方面著名人物名
  • 其他著名组织名

注：对于主办单位和竞赛举办地点或者网站没有直接关系或者依赖性的，不作为举办地点或者网站的子页面，例如Google Code Jam不应该作为TopCoder的子页面。

[编辑] 代码部分

  • Code:题库名/题目提交id
    方案一：Code_Pascal:题库名/题目提交id
    方案二：Code:题库名/题目提交id/Pascal
    方案三：Code:Pascal:题库名/题目提交id
    方案四：Code:题库名/题目提交id@Pascal（没有程序上的支持）
  • 把题目名/题目提交id换成算法、数据结构、经典题目名，命名规则同上

[编辑] 版权文章

  • Article:我的OI故事(GF版) (蒋炎岩)
  • Article:写给所有性别为男的中学生 (Freedian)
  • Article:关于演讲的若干建议 (胡伟栋)
  • Article:参考系与坐标系 (汪晔)
  • Article:浅析平面Voronoi图的应用 (王栋)

注：我写的题目说不定和原文有点出入，大家可以帮忙改一下。。

[编辑] 注意问题

  • 不要在主名字空间发起讨论或者描述相关政策
  • 不要转载作者声明禁止转载的有版权内容，转载需要注明作者和来源
  • 大家没有达成共识的情况下不要转载特定的有版权内容，尽量由自己重写并以gfdl版权发布
  • Article:以外的页面不得重定向到Article中的页面
  • Article:中的页面不得加入以算法、数据结构、题目等为主体的目录，只能通过专用的文章目录和相关页面链接
  • 以算法、数据结构、题目等为主体的导航模板不得在主要部分直接链接到Article:中的内容
