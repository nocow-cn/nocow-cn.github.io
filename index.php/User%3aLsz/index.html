[编辑] The way to Mandriva 2008.0

17:41 2008年3月2日 (CST)

Someone said Kubuntu is the WORST KDE-based OS. So I am installing Mandriva now...

I'm in Live-CD and there is no input method or Chinese font now. So the first sentences will be in English.

First, fetch a CD of Mandriva 2008.0. Then I booted with it. After clicking 'Live Install', I realized that it does not have the function of re-sizing (FAT32)partitions. Dixie! Reboot with Parted Magic and then re-size my Windows D: from 21.x GB to 13.x GB since I rarely use it. Back to Live CD. Make all the 8.x GB the root (/), ext3.

Then, when the Grub setup dialog appeared, I realized that I should backup the menu.lst in Kubuntu before it is too late...

I inserted a USB disk to save the menu.lst file. Just as I opened the USB disk in a Konqueror, something unbelievable happened:

The install guide CLOSED! It seemed that it CRASHED! I have to install it from the beginning again!

Oh! The crash of the grub installer crashed 3 times! At last, I had to install it manually. Maybe you could try to not to edit those thing, straightly press OK. It will boot Mandriva, then in a tool, you can add other OSes there. That tool won't crash often. But I prefer to edit menu.lst directly in a vi.

Back to rebooting. After reboot, choose several simple options such as network, keyboard, resolution, 3D-desktop, username, password and so on.

Then, open "Configure Your Computer" in the menu. In the first entry, you can add some media to the list. I chose one in "China" and another in "Taiwan". Then, install something like scim-pinyin, scim-bridge, skim, kde-i18n-zh, Chinese fonts from this.

Open Kcontrol from the menu and there you can add the language "Chinese" now. After that, reboot the X.

好了，现在就有中文了。至于字体美化什么的，自己看着来吧。源里面有文泉驿的点阵和矢量(正黑)字体。然后是安装fpc，到官网上去下载2.2.0版的rpm，装上。然后从源里面装上gcc,gcc-cpp和gdb什么的。如果不装gcc，fpc无法使用。为啥？因为没有ld。源里面还有eva，但是不知道为啥界面是英文的。

对于用惯了Ubuntu的人提醒一句：Mandriva默认没有sudo，只有su，su的密码是root(或者切换到的用户)的密码而不是当前用户密码。

文科和理科最根本的差别是什么？为什么我们要选择理科？我曾经跟不少人说过这样一种想法，大家都比较认同。

总的来说，文科给这个世界带来的是分裂、战争和倒退，而理科给这个世界带来的则是团结、和平和前进的动力。文科学习政治，学习历史，学习经济，学习法律，学习语言，学习哲学，学习宗教，这些东西都是国家与国家之间的文化差异，最终带来的是文化的冲突，进而导致世界的分裂和战争。但科学是不分国界的，数学符号永远是全世界统一的语言，全世界的科学家们都有着一个共同的目标——用科技的力量造福于人类。很可能只有科学才能团结整个世界，使人类进步。

我曾经思考，学习历史到底有什么用。偶然发现两句名人名言，我非常赞同：历史是胜利者的宣传，只有传记才是真实的历史。学习历史，不如展望未来。中国是一个古老的文明大国，但这个民族似乎不大会向前看，这甚至能从电影电视业上看出来：中国喜欢拍古装片，美国则擅长于科幻片。同样地，文科的学问博大精深，但多数时候只是回首过去。而理科所学的才是真正富有想象力，敢于猜想，热衷于发明创造，展望未来世界的学问。

[编辑] 强烈抵制脑残体字！

    • 070916

脑残体，脑残人士的字，就是莓兲想埝祢巳宬ｓ”１．种漝惯之类，使用偏僻字来代替常用字。 这种文字并不是通常意义上的网络语言。它与Orz、886、ls、RP以及AC等完全不是一回事。

网络语言是为了使人们交流更方便、更快捷而产生的“黑话”，而“脑残体”输入显然不方便，看起来既费眼又费解，唯一作用是炫耀。然而，这种炫耀会导致脑残体盛行，是对中国文化的摧残！

我承诺：凡是签名中含有脑残字的一律踢出好友名单！

含泪：这是外星语言吧，看不懂，简直是在糟蹋我们的汉字，坚决抵制所谓的“火星文”、“脑残体”！ 　　依风逍遥：常见到有这样的签名，还有起这样名字的网友，我ＱＱ上有这样签名的网友，立即拉入黑名单！ 　　青若：现存中国最古老的一种成熟文字－－甲骨文发现于我们河南，世界上最早的文字起源－－契刻符号发现于河南舞阳。河南可以说是汉语的发源地。作为河南人，更应该使用规范汉字，维护汉语的尊严！ 　　红薯地：美国小说家霍桑有句名言：“纯洁的良心比任何东西都可贵。”而对于一个国家和民族来说，纯洁的语言文字比任何东西都可贵。 　　我想我是海：俺觉得这样的“脑残体”已经触及了汉字的底线。其实，很多网络上的词已经被收纳进词典，也被人们广泛熟悉并使用。这说明汉语是兼收并蓄的。但是使用不规范的文字、符号，只为表达个性，折磨自己也折磨他人，这样的做法应该受到抵制。 　　剑吼西风：支持捍卫祖国语言的纯洁性，拒绝使用不规范汉字，反对“恶搞”民族语言。汉语是中华民族的根，热爱我们的国家，就要爱护我们的文字！热情倡议河南的网友们使用规范汉字，对“火星文”说不！

  • UNIX航空公司

每个人都带着飞机的一部分来到机场，大家都来到飞机跑道上，一起把飞机一片一片的拼起来，而且不断地争论者应该搭建成一架什么样的飞机。

  • DOS航空公司

起初大家都在后面推动飞机，直到，飞机开始滑行，然后再一个个的跳上飞机一同滑行，直到，飞机再次掉下来。如是反复，不断地起降……

  • Mac航空公司

　　所有的乘务员、机长、行李工和检票员看起来都一模一样。每次当你问及细节的时候，会被礼貌但是坚定地告知：你不需要知道，而且你其实并不想知道，你什么都不必知道因为一切都都已经安排妥当了，你只需要闭嘴就行了。

  • Windows(应该是<=2000, lsz注)航空公司

　　机场设施还不错，而且色彩丰富，乘务员很友好，行李托管和登机都很方便，起飞也很顺畅。只是，在空中飞行10分钟左右之后，飞机会毫无征兆的爆炸坠毁。

  • Windows NT航空公司

和Windows航空公司一样，但是更贵。使用了更大的飞机，在爆炸的时候能将方圆40里之内的其他飞机都摧毁。

  • Windows XP航空公司

你出现在机场，拿着只允许乘坐XP航空飞机的机票。所有的飞机都是一样的，颜色很显眼，而且比需要的体积大三倍。所有的标志都挺大，指向同一个方向。无论你走哪条路，都会出现一个穿着斗篷，带着尖帽子的人，让你跟随他的指引。你的行李和衣服都都被脱下去了，换上了XP航空统一的制服和箱子。在需要点餐或者饮料之前，你得跟你的旅行代理协商。在飞行途中，你需要被有规律的检查。如果你在途中上了两次以上的厕所，得重新补票。

  • Linux航空

其他操作系统航空公司中，不满意的员工准备开启自己的新航线。他们搭建飞机、售票处，而且自己造跑道。他们只收取一点点印制机票的成本，而且你还可以自己下载并打印机票。当你登上班机，会给你一个座位，几颗螺丝，一个板子和一本如何调整座位的手册。这个能够全方位调整的座位，会让你非常舒适，飞机起飞和降落都不会有问题，中途提供的餐饮也相当不错。你想告诉其他航空公司的旅客，搭乘Linux的航班很舒适，不过他们大部分会惊讶到：“那座位有什么好调整的？”

  • vista航空

这是一架巨大豪华航母级的飞机，它有着完整的设施,包含了大部份娱乐应用，上面有自动售货机，饮料吧，游戏机房等等。但是，要看你买了多少钱的门票才能确定你是否可以使用这些服务，而且如果你买了低等仓的门票，那么那此设备也就空着。 而且对你的使用者要求很高，必需要的人才能有西装革履才能在上面活动。 飞行的途中，时不时会漏风进来。而航空公司称这是一种情调，而风洞每年将会有上万个之多。
